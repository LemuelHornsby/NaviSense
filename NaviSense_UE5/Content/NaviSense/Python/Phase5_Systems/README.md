# Phase 5 — Systems helper scripts

Run these from the Unreal editor: **Tools → Execute Python Script…** with
`NaviSense_Monaco` open. Run in order the first time.

| # | Script | What it does | Writes? |
|---|--------|--------------|---------|
| 01 | `01_dump_monaco_inventory.py` | Dumps every actor in Monaco to `Saved/NaviSense_Reports/monaco_inventory.json` + a class-count summary and a system-presence checklist in the Output Log. | Read-only |
| 02 | `02_fix_yacht_materials.py` | Disables Nanite on `unity_yacht_model` to silence the ~2,700 "translucent material on Nanite" warnings and make the water/glass/plastic slots render. | Saves the mesh asset |
| 03 | `03_scene_health_check.py` | Disables World Bounds Checks (Cesium-recommended), reports Cesium tileset health (one failed to import per the log), checks georeference origin, reminds about substepping. | Toggles World Settings (save the level after) |

After running 02 and 03, do **File → Save All**.

These were generated by the next-steps automation. See
`Documents/NaviSense_UE5_Current_Master_Guide.pdf` for the full plan.
